📂 디렉토리 구조:
├── dist/            ⭐ 이곳은 프로덕션 빌드입니다 (직접 사용 권장)
│   ├── index.html   → 이 파일을 브라우저에서 열어 웹사이트를 미리보기
│   ├── assets/      → 번들된 CSS, JS 및 기타 리소스 파일
│   └── ...
├── src/             📝 소스 코드 디렉토리 (개발 및 수정용)
│   ├── components/  → React 컴포넌트
│   ├── App.jsx      → 메인 애플리케이션 컴포넌트
│   ├── index.js     → 엔트리 파일
│   └── ...
├── package.json     📦 프로젝트 종속성 구성
├── README.md        📖 프로젝트 문서
└── ...

🚀 빠른 시작 (두 가지 옵션):
옵션 1: 직접 사용 (초보자에게 권장)
1. dist/ 디렉토리로 이동
2. index.html 파일을 더블 클릭
3. 웹사이트가 브라우저에서 열립니다!

옵션 2: 개발 모드 (코딩 경험이 있는 사용자용)
1. Node.js가 설치되어 있는지 확인 (권장 버전: 16+)
2. 프로젝트 루트 디렉토리에서 터미널/명령줄 열기
3. 실행: npm install
4. 실행: npm start
5. 개발 서버가 자동으로 브라우저에서 열립니다

💡 팁:
- dist/ 디렉토리는 최적화된 빌드를 포함하고 있으며 서버에 직접 배포할 수 있습니다
- 루트 수준의 index.html은 최종 버전이 아닙니다 — 항상 dist/ 내부의 것을 사용하세요
- 코드를 수정하려면 src/ 내부의 파일을 편집하고 프로젝트를 다시 빌드하세요
- 프로젝트는 TypeScript와 Tailwind CSS 기술 스택을 사용합니다
